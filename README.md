## About The Project

The goal of this project was to generate an exam timetable which will have least clashes and bearable workload for both students and invigilators.
    
Files include:

* Code (ExamScheduler_GeneticAlgorithm_AbdullahNaveed.ipynb)
* Report
* Data

### Built With

* [Python3]
* [Jupyter Notebook]

### Prerequisites
* Jupyter Notebook
* Python3
* PrettyTable
